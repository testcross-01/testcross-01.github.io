String

  1. 不可变字符串

    String类代表字符串。 Java程序中的所有字符串文字（例如”abc” ）都被实现为此类的实例。字符串不变; 它们的值在创建后不能被更改。Java语言为字符串连接运算符（+）提供特殊支持，并为其他对象转换为字符串。 字符串连接是通过StringBuilder （或StringBuffer ）类及其append方法实现的。 字符串转换是通过方法来实现toString ，由下式定义Object和继承由在Java中的所有类。

  2. 字符串的比较

    运用==运算符对比字符串只能对比是否是同一引用（引用的地址是否相等），即使String放置于常量池当中，也可能出现相同的多个字符串的地址不同。应当运用equals方法来进行比较。 String abc=”abc”; String a=”a”; System.out.println(a.equals(abc.substring(0,1))); System.out.println(a==abc.substring(0,1));结果truefalse

  3. 代码单元和代码点char类型以UTF-16表示unicode代码单元，length方法返回UTF-16编码下的代码单元数量，大部分字符用一个代码单元表示，辅助字符用两个。实际长度调用codePointCount方法

  4. 构建字符串

    利用StringBuilder建立字符串可以提高效率，利用对象的append方法添加字符或字符串最后利用toString方法转换为字符串。
    java.lang.StringBuilder****StringBuilder一个可变的字符序列。 此类提供与StringBuffer相同的API，但不保证同步。一般用于单线程字符缓存的地方，一般使用本类，因为有较快的速度。每个字符串构建器都有一个容量。 只要字符串构建器中包含的字符序列的长度不超过容量，则不需要分配新的内部缓冲区。 如果内部缓冲区溢出，则会自动变大。
    java.lang.String主要方法：append 追加相应参数到序列中setCharAt 指定索引处的字符设为该参数insert 将参数插入到指定索引delete 删除次序列中的字符reserve 反转此序列
    java.lang.StringBufferStringBuffer线程安全，可变的字符序列。 字符串缓冲区就像一个String，但可以修改。 在任何时间点，它包含一些特定的字符序列，但可以通过某些方法调用来更改序列的长度和内容。字符串缓冲区可以安全地被多个线程使用。 这些方法在必要时进行同步，以便任何特定实例上的所有操作都按照与所涉及的各个线程所执行的方法调用顺序一致的顺序发生。有待考察编译期常量会在编译阶段存入调用类的常量池，本质上并没有直接引用到定义常量的类，因此不会触发定义常量的类的初始化.Charsequence
